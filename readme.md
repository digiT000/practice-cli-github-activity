# GitHub User Activity CLI (Roadmap.sh Practice Project)

A simple Node.js command-line interface to fetch and display recent activity for a GitHub user.

## Features

- Fetches recent events from the GitHub API.
- Aggregates activity by repository.
- Displays friendly summaries for:
  - Push events
  - Pull Requests
  - Branch/Tag creation
  - Issues
  - Watching/Starring
  - Forking

## Prerequisites

- Node.js installed on your machine.

## Usage

1. Open your terminal in the project directory.
2. Run the script:
   ```bash
   node index.js
   ```
3. Enter a GitHub username when prompted.
4. View the recent activity summary in the console.

## Example Output

```text
Enter Github username : digiT000

Recent GitHub Activity:

- Pushed 3 commit(s) to digiT000/navigation_bar_collection
- Created branch/tag in digiT000/navigation_bar_collection
- Pushed 2 commit(s) to digiT000/applikasi_cbt
```
